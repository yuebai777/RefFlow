# RefFlow 🎨

**RefFlow** (Local Reference Trainer) is a professional, offline-first gesture drawing and reference training tool tailored for artists.

> RefFlow 是一款专为艺术家打造的动态速写与临摹训练图片播放工具，专注于绘画练习，无需联网，完全本地化。

## ✨ Features (功能特性)

### ⏱️ Practice Mode (练习模式)
* **Customizable Timer (自定义计时)**:
    * Supports **"Class Mode"** (customizable stages, e.g., 10 images for 5 mins, then 20 images for 10 mins) or **"Fixed Duration"** mode.
    * 支持“课堂模式”（自由添加自定义阶段，如：五分钟x10张、十分钟x20张……）或“固定时长”模式。
* **Smart Flip (智能翻转)**:
    * Random horizontal flipping to train observation and break muscle memory.
    * 随机水平翻转图片，打破惯性思维，训练观察力。
* **Value Study (黑白滤镜)**:
    * One-click Grayscale mode for value study.
    * 一键黑白模式，帮助专注于明暗与素描关系。

### 🖥️ Immersive Experience (沉浸体验)
* **Always on Top (窗口置顶)**:
    * Pin the window to keep your reference visible over painting software.
    * 支持窗口置顶（图钉），确保参考图始终位于绘画软件上方。
* **Distraction Free (沉浸视觉)**:
    * Auto-hiding title bar and clear pause screen (no blur/overlay) for better detailing.
    * 标题栏自动隐藏，暂停时不模糊画面，最大化屏幕利用率，方便定格临摹。

### 📂 Management & Workflow (管理与工作流)
* **Progress Tracking (进度标记)**:
    * Mark images as **"Drawn"**. Visual indicators in the list show your progress, and alerts prevent repeating the same image.
    * 支持标记图片为“已画”，在列表中直观显示。练习前会自动检测并提示是否包含已画素材。
* **Library Management (素材库管理)**:
    * Delete unwanted images or remove folders directly within the app.
    * 支持在软件内直接删除不满意的图片或移除文件夹。
* **Presets & Memory (预设与记忆)**:
    * Auto-saves library and settings. Create **"Presets"** to switch between different practice configurations instantly.
    * 自动记忆素材库与设置状态。支持保存“预设”，一键切换不同的练习方案（如“每日速写”或“色彩练习”）。
* **Offline First (离线优先)**:
    * Works completely without an internet connection.
    * 无需联网，随时随地开启练习。

## 🚀 How to Use (使用指南)

1.  **Download (下载)**: Download the latest installer from [Releases](https://github.com/yuebai777/RefFlow/releases).
      * 从 Releases 页面下载最新安装包。
2.  **Install (安装)**: Install and run RefFlow.
      * 安装并运行 RefFlow。
3.  **Import (导入)**: Drag & Drop your reference folder or images into the app.
      * 直接将图片文件夹或单张图片拖拽到应用窗口中。
4.  **Configure (配置)**: Select a Preset or set up your timer and preferences.
      * 选择一个预设，或手动设置倒计时与偏好。
5.  **Start (开始)**: Start drawing! Use the "Pin" button to keep it on top.
      * 开始绘画！使用“图钉”按钮将窗口固定在最前。
